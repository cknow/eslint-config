## Install

```
npm install eslint-config-clicknow --save-dev
```

## Usage
> ESLint [Configuration File Formats](http://eslint.org/docs/user-guide/configuring#configuration-file-formats)

Add some ESLint config to your `.eslintrc.js`:

```js
module.exports = {
    extends: 'clicknow'
};
```
